### 1. Features-ka muhiimka ah
- Homepage (featured courses + latest blog posts + search).
- Koorsooyin:
  • Page koorsooyin dhan (filters: category, level, price).
  • Page koorso gaar ah (description, curriculum, instructor, reviews, buy/enroll).
  • Instructor dashboard: upload koorso (title, description, price, video, lessons).
  • Student dashboard: koorsooyin uu iibsaday, progress, downloads.
- Qayb iib (e-commerce): checkout, discount codes, invoices.
- Payment integration: Zaad, EVC+, iyo haddii la awoodo PayPal/Stripe/cards.
- Instructor payouts: admin wuxuu go’aamin karaa boqolkiiba qaybinta (70/30 ama kale).
- Blog / Maqaallo: publish, categories, tags, commenting.
- Adsense integration: boosaska xayeysiiska (header, article, sidebar), iyo pages u diyaarsan (Privacy Policy, Terms, Contact).
- SEO & Analytics: sitemap, robots.txt, GA + Search Console, schema markup (Article & Course).
- Admin Panel: users, koorsooyin, articles, ads, orders, payouts.
- Security: HTTPS, validation, file upload checks.
- Responsive Design: mobile-first, app-like UX.
- Languages: Somali + English (must).

### 2. Tech Stack (doorasho)
- Option 1 (fast build): WordPress + TutorLMS/LearnDash + WooCommerce + Adsense plugin + multilingual plugin (WPML/Polylang).
- Option 2 (custom scalable): Laravel backend + React frontend + REST API + Mobile money/Zaad integration.

Fadlan ii sheeg midka aad isticmaali doontaan.
